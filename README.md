# TurtleShip

A Pygame shooting game featuring naval combat across five stages.

## Features

- Five stages with different enemies, bosses, and attack patterns
- Player movement and shooting controls
- Score, stage, boss, and game-state management
- Resizable and maximized window modes
- Image and audio assets loaded from the `assets` directory

## Requirements

- Python
- Pygame

Install Pygame:

```bash
pip install pygame
```

## Run

```bash
python basic.py
```

Use the `--windowed` option to start without maximizing the window:

```bash
python basic.py --windowed
```

## Project Structure

- `basic.py`: Main game implementation
- `assets/images/`: Image assets
- `assets/audio/`: Music and sound effects
