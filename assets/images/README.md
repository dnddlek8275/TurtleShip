# Image Assets

Place optional image files here. PNG, JPG, and JPEG are supported.

## Common

- `splash.png`
- `main_menu.png`
- `game_background.png`
- `player.png`
- `enemy.png`
- `mini_boss.png`
- `boss_room.png`

## Per Stage

Use `N` as the stage number from 1 to 5.

- `stageN.png`
- `enemy_stageN.png`
- `mini_boss_stageN.png` or `boss_stageN.png`
- `bullet_stageN.png`
- `projectile_stageN.png`
